# Northern Shield — Game Loop

*How play flows across seconds, battles, and campaigns*

---

## Macro loop (campaign lifetime)

```
THREAT ARRIVES → PREPARE FORTRESS & WARBAND → FIGHT → AFTERMATH → INVEST & GROW → THREAT ESCALATES → …
```

The loop is a **spiral**, not a circle. Each cycle should leave the fortress **different** — visually, mechanically, or emotionally — even after a defeat.

| Phase | Player question | Fantasy |
|-------|-----------------|---------|
| **Threat** | Where is the war? | The North is not safe |
| **Prepare** | How do we hold? | I am the jarl who plans |
| **Fight** | Did the plan work? | My people execute what I chose |
| **Aftermath** | What did it mean? | Saga, scars, consequences |
| **Invest** | What do we become? | The home grows stronger |

---

## Session loop (one sitting)

```
Slot / save → Region → Command map → [War Camp] → Fortress prep → Battle → After action → (repeat or exit)
```

**War Camp** is optional mid-assault but mandatory for recruitment, deep equipment, and long-term fortress buildings. **Fortress prep** is mandatory before each assault — assignment, repairs, siege.

Screen separation (non-negotiable):

| Screen | Shows | Never shows |
|--------|-------|-------------|
| Command map | Fronts, assault intel | Battlefield grid |
| War Camp | People, meta buildings | Live combat |
| Fortress prep | Posts, repairs, schematic | Recruit shop |
| Battle | Combat execution | Build docks (campaign) |
| After action | Prose, casualties, damage | Layout editor |

---

## Battle loop (single assault)

```
INTEL → (locked plan) → WAVE 1 → WAVE 2 → … → VERDICT
```

In campaign, **deployment is locked** when the assault begins. Mid-assault building is prototype debt; target state is **execution only**.

Between waves within one assault:

- Brief breath to read casualties and gate integrity
- Auto-advance for waves 2+ (no redeploy fantasy)
- Rewards feed **battle economy** (gold) and **hero XP**

---

## Economic loop (layers)

See [economy.md](economy.md). At loop level:

1. **Battle** earns gold and immediate relief
2. **Campaign** spends wood / stone / iron / food on growth
3. **Kingdom** spends reputation and ancient knowledge on identity upgrades

The player should feel **tension between now and later**: repair the west gate or expand the forge?

---

## Progression loop (fortress tiers)

See [fortress_progression.md](fortress_progression.md) and [roadmap.md](roadmap.md).

Each fortress **phase** (Village → Settlement → … → Last Bastion) changes:

- What buildings exist
- What systems unlock
- What threats mean emotionally
- What “winning” looks like

## Attachment loop (heroes)

```
Recruit → Name → Deploy → Survive / Scar → Promote → Legend → (Legacy / Succession)
```

Death is not always erase. **Legacy**, chronicle entries, and fortress memorials keep legends in the world.

See [heroes.md](heroes.md).
