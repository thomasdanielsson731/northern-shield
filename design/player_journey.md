# Northern Shield — Player Journey

*Milestone 4 · the emotional arc of a first-time jarl*

This document describes **what it feels like** to play Northern Shield — not what systems exist. Systems are in [progression_tree.md](progression_tree.md); principles in [north_star.md](north_star.md). No new mechanics are proposed here.

**Audience:** UX, narrative, audio, art, onboarding — anyone who touches player-facing flow.

---

## Journey shape

```
FEAR → RELIEF → ATTACHMENT → RESPONSIBILITY → PRIDE → GRIEF → STEWARDSHIP → AWE → LEGACY
```

The player begins as a **survivor** and ends as a **chronicler of a civilization**. Complexity arrives as **chapters**, not tutorials.

---

## Part I — Emotional chapters

Each milestone uses the same lens: senses, cognition, learning, responsibility, memory, fortress change, player change.

---

### The First Night

| Dimension | Experience |
|-----------|------------|
| **Sees** | A tiny palisade, one fire, one figure on the wall. Command map is mostly dark. West gate highlighted. |
| **Hears** | Wind, distant howl, skald whisper: *"Hold the gate. Nothing else matters tonight."* No music swell — sparse drum. |
| **Thinks** | *"I only have one Berserker. If he falls, we fall."* |
| **Learned** | Assault = preparation → battle → aftermath. One post, one hero. |
| **New responsibility** | Assign the defender. Press Begin Assault. |
| **Memorable because** | Absolute vulnerability — the whole kingdom is one person wide. |
| **Fortress changed** | Nothing yet — but the fire is lit. |
| **Player changed** | From observer to **jarl** with one decision that matters. |

## Part III — Sensory palette by Age

| Age | Visual tone | Audio tone | UI tone |
|-----|-------------|------------|---------|
| I Survival | Mud, fire, small | Sparse drum, wind | Few words, large type |
| II Settlement | Smoke, thatch, banners | Hearth, voices | Advisor friendly |
| III Fortress | Stone, banners, siege | Masonry, war horns | Intel formal |
| IV Stronghold | Markets, flags, scars | Trade, politics | More text, still one question per screen |
| V Citadel | Rune glow, spires | Choir, dragon | Spell prep solemn |
| VI Last Bastion | Aurora, memorials | Full theme | Chronicle first, buttons second |
