Keep Northern Shield's architecture lightweight and maintainable.

 FILE RESPONSIBILITIES:
 - src/main.js                     entry point only, one import
 - src/core/renderer.js     canvas setup + DPR scaling, exports { canvas, ctx }
 - src/core/game.js             game state (gold, lives, slain, enemies, towers, bullets, particles),
                                               game loop, input handling, HUD rendering, spawning logic, wave state.
                                               This is the integration layer — intentionally large (~1400+ lines).
 - src/grid/grid.js             Grid class, CELL enum, BFS pathfinding, draw()
 - src/entities/enemy.js   Enemy class + ENEMY_TYPES + ENEMY_DEFS (all config here)
 - src/entities/tower.js   Tower class + TOWER_TYPES + TOWER_DEFS + drawSpriteFrame() helper
 - src/entities/bullet.js Bullet class (homing, trail, returns reward on kill)
 - src/assets.js                   SPRITES registry: key → { img, frameW, frameH, cols, rows, total }

 KNOWN COUPLINGS:
 - game.js imports everything — it is the intentional integration layer
 - Bullet holds a direct reference to its target enemy (alive flag guards this)
 - Tower.update() pushes to the bullets[] array passed in from game.js
 - Enemy.reward is read by bullet.js on kill and returned as update() return value
 - BUILD_ITEMS in game.js derives from TOWER_DEFS automatically (Object.values)
 - drawSpriteFrame() in tower.js reads from SPRITES (src/assets.js) — guarded by existence check

 DO NOT INTRODUCE:
 - ECS (Entity Component System)
 - Dependency injection
 - Abstract factories
 - Service locators
 - Event buses
 - Observer/pub-sub patterns
 - Any enterprise pattern

 WHEN TO CREATE A NEW FILE:
 - Only when an existing file genuinely exceeds ~250 lines of logic
 - Not for theoretical separation of concerns
 - Not for "future flexibility"
 - game.js is an exception: it is intentionally large as the integration layer

 This is a solo indie game. Optimize for iteration speed and the ability to
 understand the code after a 2-week break.
